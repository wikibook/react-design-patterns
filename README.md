# 모던 리액트 디자인 패턴 예제

《모던 리액트 디자인 패턴》에 수록된 예제 코드를 브라우저에서 확인하고, 책 원문과 프로젝트 코드의 차이를 추적하기 위한 저장소입니다.

## 실행 환경

- Node.js 22 이상 권장
- npm 10 이상

## 시작하기

```bash
npm ci
npm run dev
```

터미널에 표시되는 로컬 주소를 열면 전체 예제 카탈로그를 확인할 수 있습니다. 예제 번호나 제목으로 검색할 수 있고, 장과 프로젝트 상태로도 필터링할 수 있습니다.

## 검증

```bash
npm run check
```

위 명령은 다음 검사를 순서대로 실행합니다.

1. ESLint 검사
2. 타입 검사
3. 프로덕션 빌드

책에서 의도적으로 소개하는 안티패턴이나 실패 사례는 ESLint 경고 또는 브라우저 콘솔 메시지를 발생시킬 수 있습니다. 검증 명령은 실제 오류와 이러한 학습용 경고를 구분합니다.

## 예제 상태

- **브라우저 실행 가능**: 독자가 단독 화면에서 동작이나 결과를 확인할 수 있는 예제
- **소스 확인용**: 다른 예제가 참조하는 훅·스토어·타입·중간 컴포넌트 또는 독립 화면이 필요하지 않은 예제
- **프로젝트 파일 없음**: 책에는 있지만 이 저장소에는 구현 파일이 없는 예제

## 디렉터리 구조

```text
src/
  pages/<장>/<예제 번호>.*  # 예: 예제 6.48 → src/pages/6/48.tsx
  ExampleCatalog.tsx        # 예제 검색 및 상태 카탈로그
  example-manifest.ts       # 책과 프로젝트의 예제 대응 정보
  routes.tsx                # 독립 실행 가능한 예제 라우트
```
